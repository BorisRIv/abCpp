For Loop / For цикъл

Какво е "For" цикъл?

Циклите for в C++ се използват за многократно изпълнение на блок код, докато определено условие е вярно. Те имат следния синтаксис:


for(задаване на начална стойност; условие; промяна на стойността)
{
	оператори;
}

Основни части на "For" цикъл

1. Задаване на начална стойност

  • ● Изпълнява се само веднъж в началото на цикъла
  • ● Обикновено се използва за инициализиране на променлива за броене
  • ● Пример: "int i=0;"

2. Условие

  • ● Проверява се преди всяко изпълнение на тялото на цикъла
  • ● Ако условието е вярно, блокът код се изпълнява
  • ● Ако условието е невярно, цикълът приключва
  • ● Пример: "i<10;"

3. Промяна на стойността

  • ● Изпълнява се след всяко изпълнение на тялото на цикъла
  • ● Обикновено се използва за увеличаване или намаляване на променливата за броене
  • ● Пример: "i++" (увеличава "i" с 1)

Пример за "For" цикъл

#include <iostream>
using namespace std;
int main()
{
	for(int i=0; i<5; i++)
	{
		cout<<i<<endl;
	}
	return 0;
}

Този код извежда стойностите на i, в случая от 0 до 4 включително.


Вложени "For" цикли

Вложените "For" цикли (или For във For) са цикли, които се намират вътре в други цикли. Те са полезни за работа с двумерни масиви или за изпълнение на сложни итерационни задачи.


Пример за вложени "For" цикли

#include <iostream>
using namespace std;
int main()
{
    for(int i=1; i<=3; i++)
    {
        for(int j=1; j<=2; j++)
        {
            cout<<"i: "<<i<<, j: "<<j<<endl;
        }
    }
    return 0;
}

Първият for цикъл се изпълнява 3 пъти, а вторият се изпълнява 2*3 пъти (защото е в първия for цикъл)


Изходът на примера ще бъде:

i: 1, j: 1
i: 1, j: 2
i: 2, j: 1
i: 2, j: 2
i: 3, j: 1
i: 3, j: 2
