C++ Intro / Въведение в С++

Какво е С++?

C++ е мощен и популярен език за програмиране, който е продължение на С. Различава се от С с това, че включва обектно-ориентирано програмиране (ООП). C++ има широк спектър от приложения, като системно програмиране, софтуер за вградени системи, игри и графични приложения.


Къде можем да пишем на С++?

Съществуват много програми за компилиране на С++ код. Ако за първи път програмирате на С++, препоръчваме да използвате Code::Blocks. Когато натрупате повече опит и се чувствате, че имате нужда от по-професионален интерфейс, може да използвате Visual Studio Code или Visual Studio.


Какво трябва да зная, преди да започна да пиша код?

С++ е език с ООП, което означава, че дори една стандартна програма е базирана на обекти. Когато едва започвате да програмирате обаче, това може да е сложно. Докато не научите за обектите е важно да запомните, че следния код трябва да присъства във всяка ваша програма: 


#include <iostream>
using namespace std;
int main()
{

return 0;
}

Всяка операция в С++ завършва с ";"

Коментари, т.е. текст който не се чете като код, се отбелязва с "//".
