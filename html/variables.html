Variables / Променливи

Какво е променлива?

Променливите съдържат стойности, които могат да се променят. Те са като кутии, които съдържат даден тип информация.


Инициализация

За да знае програмата какъв вид е кутията (какъв е типа на променливата) и как се казва тя (какво е името на променливата) ние я инициализацираме. 


     int num;
//    ^   ^
//    |   |
//   тип  име

В примера създадохме променлива от тип int с име num. Тя е цяло число и можем да я извикаме с запазеното име num.

По повече от един начин можем да ѝ зададем стойност. Например: 


     int num=5;
     num = 7;

Операции

Можем и чрез следните аритметични операции да променяме стойността: 


	 num = 7+5;
	 num = 7*5;   //умножава 7 по 5
	 num = 10/5;  //дели 10 на 5
	 num = 7%5;   //дава остатъка на 7 при деление на 5
	 num = 7-5;
	 num ++;      //увеличава сегашната стойност на num с 1
	 num--;	      //намалява сегашната стойност на num с 1
	 num+=5;      //увеличава сегашната стойност на num с 5
	              // горния ред работи аналогично за другите операции

* Имайте предвид, че ако пазите стойност в int променлива, при деление ( / ) С++ заравнява надолу към най-близкото цяло число. Например 9/2 дава 4. Ако пазите стойността в double или float, би се запазила като дроб (Примера от по-рано би дал 4.5).


Типове променливи

Съществуват следните типове променливи:

Тип: Каква информация пази:
int Цели числа
char ASCII Символи (Букви например)
float Дробни числа
double Дробни числа
bool Истина или лъжа (Бинарна стойност)
string Поредица от данни (възпр. като символи)
auto Променя се спрямо подадените данни
void Празен тип (Използва се почти само за функции)
