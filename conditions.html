Conditions / Условия

Условия

Условията в С++ се усъществявят чрез условни оператори като if и else.


Оператор if

На този оператор се подава условие и ако булевата му стойност е истина, зададеният му код се изпълнява, а ако е лъжа - не. Синтаксисът му е следния:


	if (num > 9);  // примерно условие
	{
	    num=9;    // примерно действие
	}

Съществуват следните оператори за булеви изрази:


    8 > 9  // ">" проверява дали 8 е по-голямо от 9
    8 < 9  // "<" проверява дали 8 е по-малко от 9
    8 == 9  // "==" проверява дали 8 е равно на 9
    8 != 9  // "!=" проверява дали 8 НЕ е равно на 9
    8 <= 9  // "<=" проверява дали 8 по-голямо или равно на 9
    8 >= 9  // ">=" проверява дали 8 по-голямо или равно на 9
	
    (num > 3) && (num < 8)
    // "&&" дава истина само когато и двете му подадени твърдения са истина, иначе дава лъжа
    // може да се замести с "and"
	
    (num > 3) || (num < 8)
    // "||" дава лъжа само когато и двете му подадени твърдения са лъжа, иначе дава истина
    // може да се замести с "or"

Оператор else

Този оператор работи само когато има if над него и кода вътре се изпълнява само ако if-a не се изпълни. Например:


if (num > 9);  // примерно условие
{
    num=9;    // примерно действие
}
else
{
    num=12;    // примерно действие
}
