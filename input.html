Input / Въвеждане

Как се въвежда в С++?

Въвеждането в С++ най-често се случва с обекта cin и оператора ">>". Например


#include <iostream>
using namespace std;
int main()
{
	int num;
	cin>>num;
	cout<<num;
	return 0;
}

Тази програма въвежда цяло число от клавиатурата и после го извежда.
