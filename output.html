Output / Извеждане

Как се извежда в С++?

Извеждането в С++ най-често се случва с обекта cout и оператора "<<". Например:


#include <iostream>
using namespace std;
int main()
{
	int num=69;
	cout<<num;
	return 0;
}

Тази програма ще изведе стойността на променливата num, в този случай: 69.


Със cout и "<<" могат да се извеждат и конкретни низове от символи (стига да са на един ред) и те трябва да са оградени с "".

#include <iostream>
using namespace std;
int main()
{
	cout<<"Здрасти! Днес е хубав ден.";
	return 0;
}

Тази програма ще изведе изречението между кавичките точно както е написано (ако изведе странни символи, това е защото c++ не харесва кирилица, в такива случаи или променете файлът на тип UTF-8 или пишете на латиница).
