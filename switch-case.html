Switch Case

Какво е "Switch Case"?

"Switch Case" в C++ се използва за многократно взимане на решения на базата на стойността на една променлива. Тя предоставя удобен начин да замените множество "if-else" изрази, когато проверявате една и съща променлива срещу различни стойности. Нека разгледаме основната структура на "Switch Case":


switch(израз)
{
	case константа-1:
		код за изпълнение;
		break;
	case константа-2:
		код за изпълнение;
		break;
	default:
		код за изпълнение;
		break;
}

Конструкцията на "Switch Case"

1. switch(израз)

  • ● Това е началото на "Switch Case" конструкцията
  • ● "израз" е изразът или променливата, чиято стойност ще бъде проверявана
  • ● "израз" трябва да бъде от тип int, char или друг целочислен тип"

2. case константа

  • ● Всеки "case" блок съдържа стойност (константа), с която се сравнява "израз"
  • ● Ако "израз" съвпада с "константа", кодът в този блок се изпълнява
  • ● "константа" трябва да бъде уникален в рамките на един "Switch Case" и трябва да бъде от същия тип като "израз"

3. break

  • ● "break" командата прекъсва изпълнението на текущия "case" блок и прескача извън "Switch Case" конструкцията
  • ● Ако "break" липсва, изпълнението ще продължи със следващия "case" блок

4. default

  • ● Това е по желание блок, който се изпълнява, ако нито един от "case" блоковете не съвпада със стойността на "израз"
  • ● "default" блокът е полезен за обработка на неочаквани или некоректни стойности

Пример за "Switch Case"

#include <iostream>
using namespace std;
int main()
{
	int day;
	cin>>day;
	switch(day)
	{
		case 1:
			cout<<"Monday"<<endl;
			break;
		case 2:
			cout<<"Tuesday"<<endl;
			break;
		case 3:
			cout<<"Wednesday"<<endl;
			break;
		case 4:
			cout<<"Thursday"<<endl;
			break;
		case 5:
			cout<<"Friday"<<endl;
			break;
		case 6:
			cout<<"Saturday"<<endl;
			break;
		case 7:
			cout<<"Sunday"<<endl;
			break;
		default:
			cout<<"Invalid"<<endl;
			break;
	}
	return 0;
}

Обяснение на примера:

  • ● Първо се декларира и инициализира променлива "day" със стойност, нека за пример въведем "3"
  • ● "Switch Case" конструкцията проверява стойността на "day"
  • ● В този случай, "day" е равно на "3", така че кодът в "case 3" се изпълнява и отпечатва "Wednesday"
  • ● "break" командата след всеки "case" блок гарантира, че програмата няма да продължи с изпълнението на следващите "case" блокове


#include <iostream>
#include <conio.h> // за _kbhit и _getch
#include <Windows.h> // за Sleep()

using namespace std;

void gotoxy(int x, int y) {
    COORD coord;
    coord.X = x;
    coord.Y = y;
    SetConsoleCursorPosition(GetStdHandle(STD_OUTPUT_HANDLE), coord);
}

int main() {
    int x = 2;
    int y = 2;

    char player;
    cout<<"Enter a character you want to play with: ";
    cin>>player;

    system("cls"); // изчиства екрана

    while (true)
    {
        gotoxy(x, y);
        cout << ' ';

        if (_kbhit()) // проверява дали е натиснат бутон от клавиатурата
        {
            char key = _getch(); // запазва натиснатия бутон

            switch (key) {
                case 'w':
                    y--;
                    break;
                case 's':
                    y++;
                    break;
                case 'a':
                    x--;
                    break;
                case 'd':
                    x++;
                    break;
            }
        }
        gotoxy(x, y);
        cout << player;
        Sleep(100); // прави пауза Sleep(1) == 1/1000 от секундата
    }

    return 0;
}
