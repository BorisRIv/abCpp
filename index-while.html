While Loop / While цикли

Дефиниция

While циклите са цикли, които изпълняват даден код, когато подаденото им условие е изпълнено. В зависимост от условието и кода while цикълът ще изпълни този код определен брой пъти.



Синтаксис


while(<условие>) // "while" е запазена дума
{
	<код, който да бъде изпълнен>; // този код ще бъде изпълнен, само ако условието е истина
}
